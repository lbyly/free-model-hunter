# 项目概述

Free-Model-Hub 是一个统一的平台，提供对多个 AI 模型提供商的访问和管理。系统使用 Python 开发，提供了包括模型抓取、API 调用、结果存储等功能。

## 目录结构

- backend/: 核心后端代码
  - api/: RESTful API 接口
  - models/: 数据模型和仓库
  - scrapers/: 各模型提供商的抓取器
  - config.py: 配置文件
  - database.py: 数据库连接
  - main.py: 程序入口
  - scheduler.py: 调度器

- static/: 前端静态文件

- docs/: 项目文档

## 功能模块

- 模型抓取：支持多个模型提供商的抓取
- API 接口：统一的 RESTful API
- 数据库：持久化存储抓取结果
- 调度器：管理抓取任务调度

## 快速开始

1. 克隆仓库
2. 创建虚拟环境并安装依赖
3. 配置环境变量
4. 运行数据库迁移
5. 启动后端服务

## 开发流程

- 编写单元测试
- 运行测试用例
- 代码审查
- 持续集成
